Dependencies:

Software:
    SCons - The build system

    gcc for the raspberry pi - Only if you want to cross-compile for the raspberry pi.  Link here: https://github.com/raspberrypi/tools/tree/master/arm-bcm2708
    libccv - For Computer Vision.
